Algorithms and Data Structure

Milestone Three Enhancement

Artifact Description

The FunnelTrack I’ve created is a full stack web application that tracks the users’ behavior through an onboarding funnel and analyzes conversion rates at each stage. This application was created in May/June of 2026 as my Capstone. It includes a React frontend, a Node.js/Express backend, and a PostgreSQL database.

Justification for Inclusion

I selected this artifact for the algorithms and data structures category because the core of the application is built around data aggregation, sorting, and analytical computation. The specific components that I am showcasing for this project are the funnel analytic queries that aggregates user events by stage, the A/B testing that randomly puts users into groups, the conversion rate calculation that computes percentages from raw counts, and the drop off analysis that measures user loss between each funnel stage. This artifact has been recently updated by implementing an analytic engine that transforms the raw event data stored in the database and displays is on the dashboard.

Course Outcomes

The enhancement directly ties into the course outcomes of designing and evaluating computing solutions that solve a given problem using algorithmic principles. My funnel analytics query uses aggregation, grouping, and ordering algorithms to summarize individual user events into a clear stage by breakdown. The A/B testing uses a randomization algorithm that is unbiased 50/50 split between variants, which was needed to make statistically valid results.
