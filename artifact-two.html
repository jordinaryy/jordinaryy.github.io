Software Design and Engineering

Milestone Two Enhancement Narrative

Artifact Description

FunnelTrack is a full stack web application I created during CS 499 to demonstrate my skills in software design and engineering. The application tracks user behavior through an onboarding funnel, visualizes drop off rates at each stage, and includes A/B testing and exit survey functionality. It was created in May 2026 as my capstone project artifact.

Justification for Inclusion

I selected this artifact because it demonstrates a broad range of software engineering skills in a single cohesive project. The specific components that showcase my abilities include the separation of concerns between frontend and backend, the REST API architecture, reusable React component design, and the integration of multiple third party libraries. The artifact was improved by implementing a fully functional React frontend with dynamic page routing, a Node/Express backend with organized route files, and a connected PostgreSQL database. These enhancements transformed the concept into a working full stack application.

Course Outcomes

This enhancement demonstrates progress toward several course outcomes. I demonstrated the ability to use well founded techniques and tools in computing practices by implementing industry standard technologies including React, Node.js, Express, and PostgreSQL. I developed a security mindset by implementing parameterized queries across all database operations to prevent SQL injection attacks. I also designed and delivered professional quality visual communications through the dashboard UI which presents complex funnel data in a clear and accessible way.

Reflection

The process of building FunnelTrack taught me how the different layers of a full stack application communicate with each other. One of the biggest challenges I faced was resolving version compatibility issues between React 19 and third party libraries like recharts, which required me to switch to chart.js. I also learned the importance of environment variables for keeping sensitive database credentials out of my codebase. Through this process I gained a much deeper understanding of REST API design, React component architecture, and database connection management than I had from any previous course.
