Software Design and Engineering

Milestone Two Enhancement

Artifact Description

The project I am creating is called Funnel Track and it is a full stack web application. I wanted to demonstrate my own skills in software design and engineering vs doing a project that has been already created. This project initially was going to be something that SNHU would help with by giving me data from the school to do a funnel analysis on but due to confidentiality/legal reasons, I couldn’t. So, I thought it would be cool to do an application that tracks user behavior through an onboarding funnel, visualizing drop off rates at each stage and includes an A/B testing and exit survey. It’s like a big bowl of data and I’m excited to see how it pans in a real world way.

Justification for Inclusion

I selected this project because it provides a wide range of software engineering skills within a single project. The specific components that show my ability include the frontend and backend, the REST API architecture, reusable React component design, and multiple third party libraries. The artifact was improved by having a React frontend with dynamic routing, a Node/Express backend, and a connected PostgreSQL database.

Course Outcomes

This enhancement aligns with the course’s goals in mind. I demonstrated my ability to use founded techniques like React, Node.js, Express, and PostgreSQL. I developed ahead of time some security measures like parameterizing my queries to prevent SQL injection attacks.

Reflection

The process of building this project has been fun and eye opening. It taught me how each layer of the full stack process communicates with each other. One of the biggest challenges I’ve had to face was fixing a compatibility issue between React 19 and recharts (a third party library), which resulted in me switching to chart.js. I also learned the importance of .env which helps keep sensitive data safe for my database. This process has taught me more about REST API design, React, and database connection. Though the full stack class before this was very helpful going into this project.
